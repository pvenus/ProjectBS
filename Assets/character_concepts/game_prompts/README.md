# Game Prompts

This folder contains copy-ready prompts.

Use these files as the text a user gives to an agent for a concrete task.
Prompts may reference guide documents, but they should stay short and task
oriented.

Guides, schemas, SO explanations, policies, and long-form reference material
belong in:

```text
Assets/character_concepts/game_prompt_guide
```

When creating or reviewing prompts, use:

```text
Assets/character_concepts/game_prompt_guide/prompt/PromptAuthoringGuide.md
Assets/character_concepts/game_prompt_guide/prompt/PromptEvaluationGuide.md
```

Prompt evaluation copy-ready prompts are under:

```text
Assets/character_concepts/game_prompts/prompt
```

Stage node and popup event JSON prompts are under:

```text
Assets/character_concepts/game_prompts/stage
```

Strategic item JSON generation:

```text
Assets/character_concepts/game_prompts/item/StrategicItemSOJsonGeneratePrompt.md
```

Standalone strategic skill JSON generation:

```text
Assets/character_concepts/game_prompts/item/StrategicSkillSOJsonGeneratePrompt.md
Assets/character_concepts/game_prompts/item/StrategicSkillReversePlanningPrompt.md
```

Relic item JSON generation:

```text
Assets/character_concepts/game_prompts/item/RelicItemSOJsonGeneratePrompt.md
```

Do not put reference guides in this folder.
