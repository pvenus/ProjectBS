

# Character Create Guide

## Input
- Design File Name

---

## Step 1. Character Concept Image
- Open https://www.pixellab.ai/create-character
- Click **Create**.
- In the popup panel, select the **Create from text** tab.
- Select **Character Type** to match the character concept.
  - If the character is a quadruped, select the appropriate **Quadruped** model.
- Set **Generation Mode** to **Pro**.
- Write the **Character Description**.
- Set **Camera View** to **High Top-Down**.
- Adjust the image width, height, and aspect ratio as needed.
- Set **Detail** to **Highly detailed**.
- Set **Outline** to **Black outline**.
- Click **Generate v3 Character**.

---

## Step 2. Image Download
- Download the generated image.
- Save it to the project.
