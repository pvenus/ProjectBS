# 돌을 낳는 닭

산골 장터에서 한 농부가 매일 옥돌을 낳는다는 흰 닭을 팔고 있었다. 닭장 바닥에는 푸른 돌 세 알이 놓여 있었지만, 닭의 발목은 오래 묶인 듯 피가 배어 있었다. 농부는 산신의 복이라 큰소리쳤고, 곁에 선 아이는 닭이 밤마다 사람 목소리로 집에 보내 달라고 운다고 속삭였다.

선택 1: 농부에게 값을 치르고 닭을 산 뒤 발목의 끈을 풀어준다.

[유물 보상]

끈이 풀리자 닭은 깃털 사이에서 작은 청옥 조각 하나를 떨어뜨리고 평범한 산새의 모습으로 변해 날아갔다. 청옥은 굶주린 짐승의 울음이 가까워지면 미세하게 떨렸다. 농부가 자랑하던 나머지 돌들은 안료를 바른 자갈이었다.

선택 2: 아이의 증언과 닭장의 흔적을 장터 사람들에게 보여준다.

[전투]
[골드 획득]

속임수가 드러나자 농부 행세를 하던 밀렵꾼들이 새장 아래 숨긴 칼을 꺼냈다. 일행이 그들을 제압하자 장터 사람들은 빼앗긴 돈을 되찾았고, 밀렵꾼에게 걸려 있던 현상금 일부를 일행에게 건넸다.

선택 3: 기이한 장사에 끼어들지 않고 자리를 떠난다.

[아무 일도 일어나지 않음]

일행이 장터를 벗어날 때까지 농부의 호객 소리는 이어졌다. 산모퉁이를 돌기 직전, 닭장 쪽에서 아이가 우는 것과 닮은 짧은 울음이 한 번 들렸다.
