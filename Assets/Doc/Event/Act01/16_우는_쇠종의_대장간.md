# 우는 쇠종의 대장간

폐광 근처의 버려진 대장간에서는 사람이 없는데도 모루 치는 소리가 났다. 천장에 매달린 쇠종은 망치질이 울릴 때마다 아이처럼 흐느꼈고, 화덕 곁에는 광부들의 안전패를 녹여 만든 쇳덩이가 쌓여 있었다. 문밖에는 관가의 폐쇄 명령이 붙어 있었지만 먹은 아직 마르지 않았다.

선택 1: 화덕의 불을 끄고 녹고 있는 안전패를 꺼낸다.

[피해]
[유물 보상]

물을 붓자 뜨거운 증기와 쇳물이 튀어 일행이 화상을 입었다. 불길이 가라앉은 뒤 쇳덩이 속에서 여러 안전패의 이름이 겹쳐 새겨진 작은 철부적이 드러났다. 부적에서는 무너지는 돌이 가까울 때 쇠종 같은 떨림이 났다.

선택 2: 망치질 소리에 맞춰 쇠종을 세 번 울린다.

[체력 회복]

세 번째 종소리가 울리자 모루 위의 망치가 멈추고, 검댕 묻은 광부들의 그림자가 화덕 주위에 나타났다. 일행이 안전패에 남은 이름을 읽어주자 그림자들은 고개를 숙인 뒤 따뜻한 불빛으로 흩어졌다. 남은 온기가 일행의 굳은 몸과 상처를 풀어 주었다.

선택 3: 최근 붙인 폐쇄 명령의 주인을 기다린다.

[전투]

밤이 되자 광산 관리자의 사병들이 안전패와 사고 기록을 없애러 돌아왔다. 그들은 일행을 도굴꾼으로 몰아 칼을 뽑았고, 일행은 증거가 든 대장간을 지키기 위해 맞서 싸웠다.
