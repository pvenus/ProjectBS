# 산길의 세 밥그릇

사람이 다니지 않는 산길에 따뜻한 밥 세 그릇이 가지런히 놓여 있었다. 첫 그릇에는 잘 익은 밤, 둘째에는 검은 돌, 셋째에는 작은 젖니가 올라가 있었다. 가까운 나무 뒤에서는 마른기침이 들렸고, 흙바닥에는 어른 발자국 하나와 아이 발자국 셋이 서로 반대 방향으로 이어졌다.

선택 1: 밤이 놓인 밥을 나무 뒤의 사람에게 건넨다.

[체력 회복]

나무 뒤에는 아이들을 찾아 산을 헤매다 지친 어머니가 쓰러져 있었다. 일행이 밥을 데워 함께 나누자 숨어 지켜보던 아이 하나가 모습을 드러냈다. 아이는 어머니가 자신을 버린 것이 아니라 산적에게서 도망치게 하려 했음을 알고 품으로 돌아왔고, 일행도 따뜻한 밥으로 기운을 되찾았다.

선택 2: 검은 돌이 놓인 그릇을 들고 어른 발자국을 따라간다.

[전투]

발자국 끝에는 아이를 미끼로 부모의 재물을 빼앗던 산적들이 기다리고 있었다. 검은 돌은 먹이를 가져왔다는 신호였고, 일행이 그릇을 든 채 나타나자 산적들은 새로운 희생자로 여기고 덮쳐왔다.

선택 3: 젖니가 놓인 그릇을 아이 발자국이 향한 곳에 옮겨 둔다.

[유물 보상]

새벽이 되자 밥은 사라지고 작은 천 주머니 하나가 놓여 있었다. 안에는 아이들이 길을 잃지 않으려고 모아 둔 반딧불 돌이 들어 있었다. 희미한 빛은 사람이 밟은 오래된 길 위에서만 밝아졌다.
