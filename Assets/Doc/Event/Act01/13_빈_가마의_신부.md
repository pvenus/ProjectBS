# 빈 가마의 신부

안개 낀 고갯길 한복판에 붉은 혼례 가마가 버려져 있었다. 가마꾼은 없었지만 안쪽에서는 어린아이가 자장가를 부르는 소리가 났고, 발치에는 신부의 비녀와 아이의 짚신이 나란히 놓여 있었다. 가마의 문에는 밖에서 잠근 자물쇠가 달렸으며, 손잡이에는 산적들이 쓰는 검은 천 조각이 끼어 있었다.

선택 1: 자물쇠를 부수고 가마 안에 갇힌 이를 꺼낸다.

[전투]

가마 안에는 혼례복을 입힌 아이가 결박된 채 숨어 있었다. 아이를 미끼로 몸값을 뜯으려던 산적들이 안개 속에서 포위망을 좁혔고, 일행은 아이가 달아날 길을 지키며 그들과 맞섰다.

선택 2: 가마를 건드리지 않고 주변의 가마꾼 발자국을 추적한다.

[골드 획득]

발자국은 고개 아래의 버려진 초막으로 이어졌다. 그곳에서 술에 취해 잠든 가짜 가마꾼들과 여러 집에서 빼앗은 패물을 발견한 일행은 물건을 주인들에게 돌려주었다. 주인을 찾지 못한 돈 일부는 피해를 막은 사례로 받았다.

선택 3: 비녀와 짚신만 챙겨 관아에 증거로 가져간다.

[피해]

비녀를 집어 드는 순간 가마 밑에 숨겨진 쇠침 덫이 튀어 올라 일행의 손을 찔렀다. 인기척을 알아챈 산적들은 아이와 가마를 끌고 안개 속으로 달아났고, 길에는 부러진 자물쇠 조각만 남았다.
