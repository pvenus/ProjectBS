# 얼굴 없는 목각장이

산길 옆 움막에서 얼굴 부분이 비어 있는 나무 인형 수십 개가 발견되었다. 목각장이는 마을에서 잊힌 사람의 얼굴을 새겨 주면 인형이 대신 기억한다고 말했지만, 완성된 인형의 등에는 아이를 버린 집안의 성씨가 빼곡했다. 작업대 아래에서는 누군가 안쪽에서 두드리는 듯한 소리가 났다.

선택 1: 작업대 밑의 널빤지를 들어 올려 소리의 정체를 확인한다.

[특수 이벤트 추가]

바닥 아래에는 인형을 배달하던 아이가 숨어 있었다. 아이는 목각장이가 집집마다 죄를 고백하게 하려 했지만, 겁먹은 주민들이 자신을 붙잡으려 했다고 말했다. 아이는 아직 얼굴이 새겨지지 않은 인형 하나를 건네며, 보름밤에 산 아래 느티나무로 가져오면 잊힌 이름의 주인을 만날 수 있다고 했다.

선택 2: 성씨가 새겨진 인형을 가까운 마을에 가져가 공개한다.

[골드 획득]

주민들은 처음에는 목각장이의 저주라며 부수려 했지만, 인형 속에서 나온 오래된 아이 명부를 보고 침묵했다. 몇몇 집안은 감춰 온 일을 인정하고 공동 묘를 돌보기로 했다. 명부를 보존한 일행에게는 마을 공동 창고에서 여비가 지급되었다.

선택 3: 수상한 인형들을 모아 불태운다.

[전투]

불이 붙자 얼굴 없는 인형들이 작업대에서 뛰어내려 일행의 팔다리에 매달렸다. 잊힌 이름이 다시 사라지는 것을 막으려는 목각장이의 집념이 인형들을 움직였고, 일행은 타오르는 움막에서 빠져나가기 위해 길을 열어야 했다.

# 느티나무에 돌아온 얼굴

보름밤, 얼굴 없는 인형을 느티나무 아래 놓자 나무껍질 사이로 희미한 이목구비가 떠올랐다. 곧 아이를 산에 버렸던 집안의 후손들이 횃불을 들고 나타났고, 목각장이는 그들에게 인형에 이름을 새길지 아니면 과거를 다시 묻을지 선택하라고 했다.

선택 1: 남은 명부를 읽어 인형마다 잊힌 아이의 이름을 새긴다.

[유물 보상]

마지막 이름이 새겨지자 인형들은 작은 나무패로 변해 느티나무 뿌리 곁에 가지런히 놓였다. 목각장이는 이름 하나가 새겨진 조각칼을 일행에게 맡겼다. 조각칼은 지워진 글자 위를 지나면 희미한 획을 다시 드러냈다.

선택 2: 후손들이 직접 이름을 기억하고 기록하게 한다.

[체력 회복]

밤새 이어진 고백 끝에 주민들은 아이들의 이름을 새 장부와 비석에 옮겼다. 느티나무에서 따뜻한 수액 향이 퍼져 일행의 피로를 덜었고, 목각장이는 더 이상 대신 기억할 필요가 없다며 빈 인형들을 거두었다.

선택 3: 분노한 후손들에게 인형과 명부를 넘기고 물러난다.

[아무 일도 일어나지 않음]

일행이 숲을 벗어날 때 뒤에서 나무 부서지는 소리가 이어졌다. 다음 날 느티나무 아래에는 톱밥만 남았고, 되찾을 수 있었던 이름들도 다시 사람들의 침묵 속으로 사라졌다.
