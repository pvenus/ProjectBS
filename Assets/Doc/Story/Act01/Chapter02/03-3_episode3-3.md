# 유물을 알아본 유생

산울림 곡괭이를 얻은 이복산이 문서고에서 김도윤을 영입하고, 유물에 새겨진 광부장 이름을 원본 기록과 연결하는 합류 에피소드.

문서고에서 김도윤은 검은 곡괭이 자루의 이름들을 보고 오래된 광산 명부를 펼쳤다. 이름은 모두 붕괴 위험을 경고했다가 해고되거나 죽은 광부장들이었다. 곡괭이가 울릴 때마다 명부의 지워진 글씨가 금빛 먼지 위로 떠올랐다.

김도윤은 이것이 금을 캐는 보물이 아니라 묻힌 기록을 드러내는 유물이라고 말했다. 감독관이 없애려는 원본 명령서를 찾기 위해 그는 이복산과 동행하기로 했다.

이복산은 김도윤과 산울림 곡괭이의 이름을 광산 명부와 대조한다.

두 사람은 유물이 가리킨 폐수 운반로로 향했다. 그곳에서는 짧은 총성이 구조 신호처럼 두 번 울렸다.
